
# HTML Cards Toolkit

Toolkit para padronização e criação de cards HTML no Power BI utilizando o visual HTML Content Lite.

## 🎯 Objetivo

Fornecer funções reutilizáveis em DAX para construção de componentes visuais em HTML, permitindo:

* Padronização de layout
* Reutilização de código
* Escalabilidade de dashboards
* Facilidade de manutenção

## 📦 Estrutura

* `src/` → Funções DAX (core do toolkit)
* `docs/` → Documentação de uso
* `examples/` → Exemplos práticos

## 🧩 Componentes disponíveis

* `_HTML.CARD` → Card simples
* `_HTML.CARD.MULTI_LABEL` → Card com sub-métricas
* `_HTML.CARD.MULTI_LABEL.DETAIL` → Card com variação e badge
* `_CSS.CARD.STYLE` → Estilo padrão

## 🚀 Como usar

1. Copie as funções da pasta `src`
2. Cole no seu modelo Power BI
3. Utilize no visual HTML Content Lite

## 🏢 Organização

Este repositório faz parte do ecossistema de toolkits da organização **roqt-pbip**, com foco em padronização de soluções analíticas.
